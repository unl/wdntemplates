Demo: Grid System

The 2011 Iterative Navigation templates came with a new grid system. This allows for easier-to-implement, more flexible layout options.

The new grid system should be used for all layouts. The old grid system, while still available, is in the process of being deprecated. It is displayed below only as reference.

To implement, use the class names from below. Ex: <div class="grid6">. For the first column in a row (the left-most column), you must add the class first. Ex: <div class="grid6 first">
